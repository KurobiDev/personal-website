Private NAS Server

A tiny project I did in a day.
It's a simple NAS I set up on a Raspberry Pi, running on a Samba server.
I intend to store mainly saved memes as well as text projects.
I plan to expand the storage size of the server, as well as add a streaming service, that will allow me to stream audio and video on it.

Specs:

  • Machine: Raspberry Pi 3B+
  • OS: Raspberry Pi OS (Based on Debian 13 "Trixie")
  • Current Storage: 14 GB (Single USB Stick)
