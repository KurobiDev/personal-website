Source code of my website.

- consists mainly of `.html`, `.css` and `.js`

- developed using [eleventy](https://www.11ty.dev/)

---

**Folder structure**

- `content`: templates

- `public`: static site generated from templates in `content` folder

- `public/assets`: CSS stylesheets, JS scripts and images used in site.

---

**Running the project**

Run `npx @11ty/eleventy --serve` to start the eleventy server - you should have the site hosted at http://localhost:8080/
