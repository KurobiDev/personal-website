Private NAS Server

A tiny project I did in a day.
It's a simple NAS server I set up on a Raspberry Pi, running on a Samba service.
I intend to store mainly some saved memes as well as text projects.
I have plans to expand the storage of the NAS server, making it serve it's purpose for more, than just storing memes and documents.

Specs:

  • Machine: Raspberry Pi 3B+
  • OS: Debian 13 "Trixie" (Raspberry Pi OS)
  • Current Storage: 16 GB (Single USB Stick)
